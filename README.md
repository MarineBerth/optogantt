# OptoGantt

**Beautiful, interactive Gantt charts for Java web applications.**

OptoGantt is a commercial Java Gantt chart component library with native editions for four of the most widely used Java web frameworks. Drop it into your project and your chart is live — no JavaScript dependencies, no iframes, no compromises.

---

## Live Demo

Explore the fully interactive demo at **[gantt.nz](https://gantt.nz)** — add your own bars, adjust the date range and period, and see all features in action before you commit to anything.

---

## Features

- Interactive filter pane — adjust date range from 60 minutes to 52 weeks
- "Now" indicator stripe marking the current moment
- Hover tooltips on bars
- Colour theming via a clean API
- Hyperlinks on bars
- i18n via properties files
- Weekend highlighting
- Responsive column layout
- No JavaScript dependencies
- No iframes

## Getting Started

Download `optogantt-1.0.0.zip` from **[Gumroad](https://optomus.gumroad.com/l/yjyusa)**, unzip, and follow the quick-start guide for your framework. Each guide walks you through installation, bar construction, filter pane wiring, and optional customisation in straightforward numbered steps.

## Origin

OptoGantt began life as a berth planning component inside [MarineBerth](https://marineberth.com) — a port and marine terminal operating system built on Apache Tapestry. It proved too useful to keep locked inside an enterprise application, so it was extracted, generalised, and made available to the wider Java community.
